Mi Web Personal – IPoP

Este proyecto es una página web personal diseñada para presentar mi perfil académico, profesional y personal. Todo el contenido del sitio está integrado en un único archivo principal: miweb.html.
Contenido del sitio

 miweb.html Contiene todas las secciones de la web:  
  - Perfil Personal (presentación, foto, DAFO, competencias)
  - Perfil Formativo (estudios, formación, competencias técnicas)
  - Perfil Profesional (salidas laborales, especialización, ofertas, correo)
  - Soy Pro (mi visión profesional y forma de trabajar)
  - Currículum (CV digital + descarga en PDF)
  - estilos.css (Archivo de estilos que da coherencia visual a toda la web).  

Objetivo del proyecto

El propósito de esta web es mostrar de forma clara y organizada mi identidad personal, académica y profesional. Funciona como un portafolio básico que puedo compartir para estudios, prácticas o primeros contactos laborales.
