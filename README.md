#  Mi Web Personal – IPoP

Este proyecto es una página web personal diseñada para presentar mi perfil académico, profesional y personal. Está organizada en varias secciones independientes, cada una con su propio archivo HTML, y sirve como una carta de presentación clara y sencilla.

## Contenido del sitio

- **index.html**  
  Página principal del sitio. Funciona como portada y punto de acceso a las demás secciones.

- **perfil-personal.html**  
  Describe quién soy, mis intereses, mi contexto y mi forma de trabajar.

- **perfil-formativo.html**  
  Resume mi trayectoria educativa, estudios previos y formación actual.

- **perfil-profesional.html**  
  Presenta mis habilidades, orientación laboral y capacidades técnicas.

- **soy-pro.html**  
  Explica mi visión profesional, mis metas y mi actitud hacia el trabajo.

- **estilos.css**  
  Archivo de estilos que da coherencia visual a todas las páginas.

## Objetivo del proyecto

El propósito de esta web es mostrar de forma clara y organizada mi identidad personal, académica y profesional. Es un portafolio básico que puedo compartir para estudios, prácticas o primeros contactos laborales.
